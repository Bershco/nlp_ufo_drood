# Geographic Boundary Reference

`natural_earth_admin0_countries.geojson` is the Natural Earth 1:110m Admin 0 Countries dataset. It is stored locally so the static UFO/UAP map can draw world and United States outlines without an internet connection.

- Source: Natural Earth Vector, version 5.1.1
- Project page: https://www.naturalearthdata.com/downloads/110m-cultural-vectors/110m-admin-0-countries/
- Repository source: https://github.com/nvkelso/natural-earth-vector
- License: public domain
